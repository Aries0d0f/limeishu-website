李 梅 樹 は 早 年 日 本 に 留 学 し 、 東 京 美 術 学 校 （ 現 東 京 芸 大 の 前 身 ） に 入 り 、 １９３４ 年 に 卒 業 、 帰 台 し て 、 友 達 と 協 力 し て 台 陽 美 術 協 会 を 創 設 し ま し た 。 こ の こ と は 台 湾 の 画 壇 に 大 規 模 を 美 術 団 体 を 一 つ 増 設 し 、 画 家 達 に 自 分 の 作 品 を 発 表 で き る 新 天 地 を 開 き 、 四 十 餘 年 来 、 台 湾 美 術 界 の 新 し き 発 足 と 勃 興 に 対 し て 、 不 滅 の 功 績 を 残 し ま し た 。

    光 復 の 前 後 約 二 十 年 間 に お い て 、 李 梅 樹 は 絵 画 に 從 事 す る 一 方 、 地 方 の 建 設 に 対 し て も 多 大 を 貢 献 を し ま し た 。 例 え ば 、 日 本 統 治 時 代 の 協 議 会 員 、 戦 後 の 鎮 長 、 鎮 民 代 表 主 席 、 農 会 理 事 長 ( 農 業 協 同 組 合 委 員 長 ) 、 台 北 県 議 員 等 の 職 に 就 き 、 大 衆 の 為 に 献 身 的 に 奉 仕 し 、 そ の 廉 潔 に し て 誠 実 無 私 の 胸 襟 は 更 に 深 く 、 地 方 の 人 人 か ら 称 賛 、 尊 敬 さ れ ま し た 。

    民 国 五 十 三 年 （一 九 六 四 年）の 七 月 、 施 翠 峰 氏 は 当 時 国 立 芸 専 の 校 長 に 李 梅 樹 教 授 を 推 薦 し 、 彼 は そ の 美 術 科 主 任 に 招 聘 さ れ ま し た 。 在 任 中 、 彼 は 意 欲 的 に 教 学 の 革 新 を 図 り 、 更 に 多 く の 美 術 教 育 に 関 す る 新 知 識 を 求 め て 、 民 国 五 十 七 年 （ 一 九 六 八 年） 秋 、 李 教 授 は 教 育 部 （ 文 部 省 ） に 要 請 し 、 彫 塑 科 を 設 立 し ま し た 。 こ れ に よ り 、 台 湾 の 彫 塑 教 育 に 新 紀 元 を 開 き 、 彫 塑 人 才 の 育 成 に 深 大 を 影 響 を 及 ぼ し ま し た 。

    そ の 後 、 民 国 六 十 二 年 （一 九 七 三 年 ） に は 、 国 内 に お い て 油 絵 の 創 作 と 作 品 発 表 を 促 進 す べ く 、 画 友 と 共 に 、 中 華 民 国 油 絵 学 会 を 創 立 し 、 継 続 的 に 全 国 油 絵 展 を 開 き 、 油 絵 の 正 統 を 風 格 を 奨 励 し 、 青 年 達 を 鼓 舞 激 励 し ま し た 。

    そ れ か ら 、 民 国 六 十 七 年 （一 九 七 八 年） に は 、 中 国 美 術 協 会 理 事 長 に 就 任 し 、 常 に 全 国 的 な 美 術 活 動 を り 一 ド し 、 絶 え ず 政 府 に 、 文 化 建 設 の ピ ツ チ を 上 げ て 、 美 術 教 育 の 次 元 を 高 め る べ き だ と 呼 び か け 、 わ が 国 の 美 術 教 育 及 び 活 動 に 貢 献 す る 処 が 多 多 あ り ま し た 。

    し か る に 、 芸 術 家 及 び に 美 術 教 育 家 と し て の 外 に 、 李 梅 樹 に は 多 く の 傑 出 し た 業 績 が あ り ま す 。 特 に こ こ で 述 べ な く て は な ら な い こ と は 、 民 国 三 十 六 年（ 一 九 四 七 年 ） よ り の 三 峡 祖 師 廟 の 再 建 工 事 の 責 任 者 と し て の 仕 事 で あ り ま す 。 彼 は 、 三 峡 に 歴 史 的 価 値 を 有 す る 美 の 極 致 を 表 現 す る 廟 宇 の 建 立 に 、 全 精 魂 を 打 ち 込 ん で 企 画 、 設 計 し 、 全 国 各 地 か ら 最 優 秀 の 名 匠 た ち を 網 羅 し 、 『 美 の 殿 堂 』 を 完 成 し よ う と 現 在 尚 も 頑 張 り つ づ け て い ま す 。 三 峡 祖 師 廟 に は 、 彼 の 崇 高 な 芸 術 理 念 と 宗 教 へ の 敬 虔 さ が 融 合 し て い ま す 。 こ の 彼 の 半 生 涯 を 費 し た 華 麗 な 神 殿 が 、 今 で は 三 峡 を 台 湾 北 部 の 観 光 要 地 の 一 つ と な ら し め た の も 、 道 理 が な い で は あ り ま せ ん 。

    絵 画 に 從 事 し て 半 世 紀 以 上 に も な る 李 梅 樹 教 授 は 、 個 展 を 四 十 数 年 前 に 只 一 回 の み 開 い た の で あ り ま し た 。 絵 画 に 対 す る 態 度 が 彼 の 如 く 、 慎 重 に し て 謙 虚 な 画 家 は 、 台 湾 画 壇 に お い て 、 実 に 数 多 く は 見 ら れ ま せ ん 。 李 教 授 は 、 六 十 餘 年 の 絵 画 の 生 涯 に お い て 、 未 だ 曽 て 一 日 も 自 分 の 業 績 に 満 足 し た こ と が あ り ま せ ん 。 作 品 の 大 小 に 拘 ら ず 、 全 力 を 傾 注 し 、 そ れ 故 、 絵 画 に お い て 独 特 の 風 格 を 形 成 し ま し た 。 彼 の 絵 は 、 色 彩 豊 富 に し て 澄 明 で あ り 、 写 実 の 中 に 濃 厚 な 詩 情 と 感 性 を 帯 び 、 作 品 を 見 る 度 に 、 フ ラ ン ス の 自 然 主 義 の 大 家 コ ロ 一 （C o r o t） の 風 韻 と 趣 き を 思 い 出 し ま す 。 し か れ ど も 、 二 人 の 筆 調 は 全 く の 同 じ で は あ り ま せ ん .............

    深 く 欧 洲 や 日 本 の 画 風 に 影 響 を 受 け て い る 台 湾 の 画 壇 に お い て 、 李 梅 樹 教 授 の 芸 術 は ひ と き わ 目 立 つ て い ま す 。 彼 は 一 生 名 利 に は 非 常 に 淡 泊 、 金 銭 は 彼 に と つ て 、 永 久 に 芸 術 の 価 値 を 超 越 し 得 ず 、 幾 多 の 絵 画 収 集 家 や 画 廊 経 営 者 が お 金 を ひ き 提 げ 、 ア ト リ エ ま で 押 し か け て も 、 そ の 都 度 、 彼 の 『 作 品 は 未 完 成 だ か ら ........... 』 と い う 理 由 で 断 わ ら れ ま し た 。 彼 に と つ て 、 ど の 小 さ な 一 枚 の 絵 も 金 銭 で は 換 え ら れ な い 生 命 で あ り 、 絵 を 売 る こ と に 、 丁 度 自 分 の 霊 魂 を 売 る 様 な 苦 痛 を 感 じ る の で す 。 そ れ ほ ど 自 分 の 作 品 を 熱 愛 し て 創 作 し て 来 ま し た ............
