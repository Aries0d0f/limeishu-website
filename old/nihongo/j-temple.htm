三 峡 祖 師 廟

三 峡 祖 師 廟 の 歴 史

    　 三 峡 は 三 つ の 河 川 に よ っ て 沖 積 さ れ た 三 角 洲 に 位 置 し 、 古 く は 三 角 湧 と 呼 ば れ て い ま し た が 、1920 年10 月1日 、 台 湾 総 督 府 に よ り 三 峡 と 改 め ら れ ま し た 。

    　 清 水 祖 師 は 俗 名 を 陳 昭 應 と 言 い 、 北 宋 の 都 、 開 封 の 祥 符 県 の 人 で 、 宋 の 国 を 守 る た め に 元 の 兵 と 戦 っ た 英 雄 で し た 。 後 に 明 の 太 祖 は 陳 昭 應 を 追 慕 し て 彼 の 魂 を 「 護 国 公 」 に 封 じ 、 福 建 省 安 渓 県 清 水 巌 に 祠 を 建 て て 祭 り ま し た 。 そ れ 以 来 、 安 渓 の 人 々 は 護 国 公 を 祖 師 公 と 呼 ぶ よ う に な り 、 祠 も 「 祖 師 廟 」 と 呼 ば れ 、 広 く 信 仰 さ れ る よ う に な り ま し た 。 陳 昭 應 は 生 前 清 水 巌 に 隠 居 し て い た の で 、 清 水 祖 師 と も 呼 ば れ て い ま す 。

    　 三 百 年 あ ま り 前 、 安 渓 の 人 々 が 清 水 祖 師 を 守 り 神 と し て 、 鄭 成 功 と と も に 海 を 渡 っ て 台 湾 へ き ま し た 。 後 に 彼 ら は 故 郷 の 安 渓 に 非 常 に よ く 似 て い る 三 峡 を 発 見 し 、 こ こ に 家 を 建 て て 開 墾 を 始 め ま し た 。 清 代 の1769年 、 三 峡 の 人 々 は 三 峡 渓 の 西 岸 に 祖 師 廟 を 建 立 し 、 廟 の 名 を 長 福 巌 と 命 名 し 、 清 水 祖 師 公 を 祭 り ま し た 。 そ し て 霊 験 あ ら た か な 清 水 祖 師 は 三 峡 の 人 々 の 心 の 支 え と な っ た の で す 。

    祖 師 廟 の 三 度 に わ た る 再 建

    　 一 度 目 の 再 建 ： 道 光13年 （1833 年 ） に 台 湾 北 部 を 襲 っ た 大 地 震 で 祖 師 廟 の 一 部 が 損 壊 し た た め 、 地 元 の 人 々 が 資 金 を 出 し 合 っ て 祖 師 廟 を 修 復 ・ 増 築 し ま し た 。

    　 二 度 目 の 再 建 ：1895年 の 下 関 条 約 に よ っ て 台 湾 が 日 本 に 割 譲 さ れ 、 日 本 軍 が 三 峡 に 進 駐 し た 際 、 地 元 の 人 々 は 祖 師 廟 を 指 令 部 と し て 抗 日 の 戦 い を 繰 り 広 げ た た め 、 日 本 軍 の 攻 撃 を 受 け て 破 壊 さ れ 、1899 年 に 再 び 人 々 が 資 金 を 集 め て 再 建 し ま し た 。

    　 三 度 目 の 再 建 ： 第 二 次 世 界 大 戦 で 日 本 は 敗 戦 し 、 台 湾 が 中 華 民 国 に 接 収 さ れ た 頃 、 長 年 手 入 れ を さ れ て い な か っ た 祖 師 廟 の 傷 み は ひ ど く な っ て い ま し た 。 そ の 頃 、 兵 隊 と し て 海 外 へ 送 ら れ て い た 三 峡 の 若 者 が 無 事 に 戻 っ て き た の は 、 祖 師 廟 の ご 加 護 の お か げ だ と 考 え た 地 元 の 人 々 が 再 建 を 決 意 し ま し た 。 こ の1947 年 か ら の 再 建 築 が 始 ま っ て か ら 今 日 ま で 、 す で に50 年 近 く が た っ て い ま す が 、 ま だ 完 成 し て い ま せ ん 。 当 初 再 建 築 の 指 揮 を 取 っ た の は 、 芸 術 専 科 学 校 の 美 術 ・ 彫 刻 主 任 で あ っ た 李 梅 樹 教 授 で し た 。 李 教 授 は 、 祖 師 廟 を 中 国 の 歴 史 ・ 文 化 ・ 寺 廟 芸 術 を 総 合 さ せ た 建 築 物 と し て 再 建 す る た め に 、 亡 く な る ま で の36 年 間 、 費 用 も 時 間 も 惜 し ま ず に 全 身 全 霊 を 傾 け て 祖 師 廟 再 建 に 取 り 組 み ま し た 。 今 日 、 祖 師 廟 を 訪 れ る 人 々 は 、 素 晴 ら し い 彫 刻 や 建 築 に 目 を 見 張 り 、 李 教 授 や 職 人 た ち の 注 い だ 心 血 に 心 を 動 か さ れ ま す 。 そ し て 祖 師 廟 は 台 湾 で は 知 ら な い 人 の い な い 「 東 方 芸 術 殿 堂 」 ─ ─ 彫 刻 博 物 館 と さ れ る よ う に な っ た の で す 。
